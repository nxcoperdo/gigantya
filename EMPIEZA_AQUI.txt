⚡ COMIENZA EN 15 MINUTOS
================================================================================

1. BASE DE DATOS (3 min)
   └─ Abre MySQL y ejecuta:
      mysql -u root -p < database/schema.sql

      Crear usuario:
      mysql -u root -p << EOF
      CREATE USER 'gigantya_user'@'localhost' IDENTIFIED BY 'gigantya123456';
      GRANT ALL PRIVILEGES ON restaurante_pedidos_gigantya.* TO 'gigantya_user'@'localhost';
      FLUSH PRIVILEGES;
      EOF

2. BACKEND (4 min)
   └─ cd server
      npm install
      npm run dev

3. FRONTEND (4 min - otra terminal)
   └─ cd client
      npm install
      npm run dev

4. ACCEDE A LA APP
   └─ Frontend:   http://localhost:5173
      Backend:    http://localhost:5000/api
      Database:   localhost:3306

================================================================================
❓ PROBLEMAS COMUNES
================================================================================

"Port 5000 already in use"
  └─ netstat -ano | findstr :5000  (Windows)
  └─ lsof -ti:5000 | xargs kill -9 (Mac/Linux)

"Cannot connect to MySQL"
  └─ Verifica que MySQL está corriendo
  └─ Verifica credenciales en .env

"npm: command not found"
  └─ Instala Node.js desde nodejs.org

"Cannot find module"
  └─ run: npm install en la carpeta
  └─ run: rm -rf node_modules package-lock.json && npm install
